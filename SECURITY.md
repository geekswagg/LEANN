# Security Policy

## Supported Versions

We currently provide security updates for versions 5.1.x and 4.0.x. Older versions are not supported.

| Version   | Supported          |
| --------- | ------------------ |
| 2.3.0     | :white_check_mark: |
| 2.2.1     | :white_check_mark: |
| 2.2.0     | :x:                |
| < 2.2.0   | :x:                |

## Reporting a Vulnerability

To report a security vulnerability, please email us at [security@example.com](mailto:security@comwenga.com) with details of the issue.

We aim to acknowledge all vulnerability reports within 3 business days. You will receive updates on the status of your report at least once a week until the issue is resolved or closed.

After reviewing your report, we will notify you whether the vulnerability has been accepted, declined, or requires further information. If accepted, we will work with you to coordinate disclosure and remediation.
